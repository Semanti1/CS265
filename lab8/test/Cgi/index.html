    To do

    Write a script to gather and validate information.

    The script will gather the following information:

    • Name
    • Address:
      • Street 1
      • Street 2
      • City
      • State
      • zip
    • Phone #

    The script will present a form, for this info. The form will call the same script again, to validate the data. The following must be filled in:

    • Name
    • Street 1
    • City
    • State
    • Zip
    • Phone #

    Further, the zip code must be in one of the 2 following forms:

    • ddddd
    • ddddd-dddd

    If any data doesn't check out the form is redisplayed with an error msg at the top, and w/the data that checked out still populating the appropriate fields.

    If all the data was okay, then simply write it back to the client in column format.

    Misc

    Generally, the form should have a nice layout, be simple, etc. Painful or difficult forms will cost you some points.
